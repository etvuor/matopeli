DiscordBotti luotu soittamaan musiikkia discord-kanavilla !play-komennolla, pysähtymään !stop-komennolla. Botissa myös muita ominaisuuksia:

!ping-komenta palauttaa chattiin viestin pong
!apina-komento palauttaa chattiin youtube-linkin kappaleesta apinaa koijataan (kummeli)
Botti myös asettaa roolit uusille servulle liittyjille ja toivottaa heidät tervetulleeksi welcome-channelissa.

Toimiakseen täytyy ladata moduulit discord.js, ffmpeg-static, yt-search, ytdl-core (npm install ffmpeg-static yt-search ytdl-core discord.js)
main.js täytyy lisätä oman applicationin token (tietoturvan vuoksi ette nää minun tokeniani)
